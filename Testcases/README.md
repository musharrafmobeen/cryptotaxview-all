# Getting started

#### For Windows
=======
## For Windows
- `npm install`
- `set CYPRESS_web_address=http://192.168.18.52:7010`
- `.\node_modules\.bin\cypress open`


#### For Linux/Macos
=======
## For Linux/Macos
- `npm install`
- `export CYPRESS_web_address=http://192.168.18.52:7010`
- `./node_modules/.bin/cypress open`
